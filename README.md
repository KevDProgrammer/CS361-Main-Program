# Accord Collection

Welcome to Accord Collection!

This is a game-library that acts as a game catalog. Made by a gamer, for gamers, who doesn't love to learn more about the games that you love? 
Maybe even games you never heard of before. It's interesting! This program features...

- A Home page with all the (currently) available titles
- A Help Menu page that shows some basic info
- A Navigation Bar (Home tab)
- Dedicated Game Pages for every titles (contaning images, music, rating and discription)
- Dynamic UIs with sound effects and hover features, because why not? (Keeps things fun)

...and the four microservices that are also included, enriches it further by adding...

- Search capability (by title or genre (use '#: ___" for genre)
- Music (OSTs) Snippets playback on every dedicated game page (play/stop/shuffle)
- Image Gallery that allows users to scroll through previews, snapshots, gameplays, cinematic shots, and more...
- Ratings are assigned to every titles, and can be found individually on each game page, or be sorted in highest-to-lowest for every available titles

## Instruction on How to Run:
- Run the four microservices first
- Then run the flaskServer (connect backend to a frontend server)
- Retrieve local host link from flaskServer terminal
- Paste in browser
- Enjoy
